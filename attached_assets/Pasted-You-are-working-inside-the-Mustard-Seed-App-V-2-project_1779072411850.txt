3. User completes the orientation
4. User completes the Five Heartbeats assessment
5. Assessment results are saved
6. User can create a goal
7. Goal saves correctly
8. Goal connects to a personal WHY

10. User can create a small daily promise or action
11. User can interact with Jai
12. Jai can reference the user’s goal, why, assessment, and progress
13. User can record progress
14. Progress saves to the correct day
15. Memory/log entries are created correctly
16. Water is awarded correctly when valid progress is recorded
17. Seed/tree/growth visual updates from real progress data
18. Calendar or daily view reflects the recorded action if this feature currently exists

20. User can return later and still see saved data

This full journey is the core accountability loop.
